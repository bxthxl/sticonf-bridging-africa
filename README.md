STIConf Bridging Africa is a digital platform created to support the organization and promotion of the **Science, Technology and Innovation Conference (STIConf)** initiative.

The platform provides an online presence for the conference, helping to connect **innovation, technology, entrepreneurship, and collaboration across Africa.**

---

# Project Overview

The STIConf platform is designed to support conference activities such as:

- Event promotion
- Information dissemination
- Participant engagement
- Technology collaboration

The application is built using modern web technologies to ensure **scalability, performance, and a responsive user experience.**

---

# Key Features

- Conference information portal
- Event and initiative promotion
- Modern responsive UI
- Backend services powered by Supabase
- Scalable React web architecture

# Installation

Clone the repository

```bash
git clone https://github.com/bxthxl/sticonf-bridging-africa.git
```

Navigate to the project directory

```bash
cd sticonf-bridging-africa
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
